## Task

Create a `BiometricGate` React component that enforces biometric authentication before rendering the application. Component must support dev mode bypass, graceful degradation for machines without biometrics, and proper error handling.

## Deliverables

✅ **src/components/BiometricGate.tsx** — State machine component with loading/error/no-biometry states  
✅ **src/main.tsx** — Wrapped `<RouterProvider>` inside `<BiometricGate>` (inside `<ThemeProvider>`)  

## Architecture

```
<QueryClientProvider>
  <ThemeProvider>
    <BiometricGate>       ← Auth gate here
      <RouterProvider />  ← App renders after auth
    </BiometricGate>
  </ThemeProvider>
</QueryClientProvider>
```

## Key Decisions

- **Dev Mode Bypass:** Check `import.meta.env.DEV`, skip auth if true
- **Graceful Degradation:** If biometryType === 0, show warning but allow access
- **Dynamic Import:** Safely handles environments without `window.__TAURI__`
- **Theme Integration:** Gate inside ThemeProvider so auth UI respects theme preference

BiometricGate component fully implemented with proper state handling, error recovery, and development mode support.
